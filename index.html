1.1. 'git --version'

Exibe a versão do git que está instalada e wm execução.

1.2. 'git config'

Serve para configurar algumas informações que serão SEMPRE colocadas nos commits que serão efetuados no repositório local.

1.2.1 'git config --global user.name "Aldo de Jesus Freire Araujo"'

Configura o Git de TODOS os repositórios locais com a informação do Usuário que efetuou os commits.

1.2.2 'git config --global user.email "aldoaraujo2000@yahoo.com"'

Configura o Git do TODOS os repositórios locais com a informação do eMail do Usuário que efetuou os commits.

1.2.3 'git config user.name "Aldo de Jesus Freire Araujo"'

Configura o Git para ESTE repositório local com a informação do Usuário que efetuou os commits.

1.2.4 'git config user.email "aldoaraujo2000@yahoo.com"'

Configura o Git para ESTE repositório local com a informação do eMail do Usuário que efetuou os commits.

1.3. 'git init'

Inicializa o repositórilo local do Git.

1.4. 'git status'

Exibe o 'status' do repositóro do local do git

1.5. 'git add'

1.5.1. 'git add nome-do-arquivo'

Coloca o arquivo de nome 'nome-do-arquivo' (tendo ele sido alterado ou excluído) no monitoramento do Git.

1.5.1. 'git add .'

Coloca TODOS os arquivos alterados ou excluídos no monitoramento do Git.

1.6. 'git log'

Exibe todos os 'commits' que foram efetuados.

Exibe qual é a 'branch' ativa neste momento.

git remote add origin https://github.com/aldoaraujo2000/Estudo-GIT-GITHUB.git
